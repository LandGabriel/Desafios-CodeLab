# Página de Login

## Sobre o Desafio 👨🏻‍💻
O objetivo deste é criar uma página de logina. O layout foi desenvolvido replicando o design fornecido no Figma.

## Figma do Projeto

Os usuários conseguem:

- Visualizar o estado de hover nas imagens dos produtos.
