# Blog Programação

## Sobre o Projeto 📝

Este projeto é um blog dedicado a artigos sobre programação. O objetivo principal é fornecer um espaço onde usuários possam encontrar e explorar conteúdos relevantes e atualizados sobre diversas áreas da programação.

## Layout do Projeto 🎨

O layout do blog foi construído com base em um design fornecido no Figma. O foco foi replicar fielmente a estrutura visual e funcionalidades, garantindo uma experiência consistente para os usuários.

### Figma do Projeto
- [Link para o Figma](https://www.figma.com/design/Yb9IBH56g7T1hdIyZ3BMNO/Desafios---CodeLab?node-id=3725-2&t=80quUA373yMQD7Ne-0)

## Funcionalidades 🚀

Os usuários podem:
- Visualizar um layout responsivo adaptado para diferentes dispositivos.
- Navegar facilmente por artigos recomendados e detalhes de contato.
- Interagir com elementos, incluindo estados de hover para imagens e botões.

## Tecnologias Utilizadas ⚙️

O projeto foi desenvolvido utilizando:
- HTML
- CSS
- JavaScript (opcional para funcionalidades adicionais)
